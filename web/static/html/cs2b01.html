¿Cuál es el propósito del curso?

El curso tiene varios propósitos, los cuales son:

  1. Diseñar, implementar y evaluar un sistema, proceso, componente o programa computacional para alcanzar las necesidades deseadas.
  2. Trabajar efectivamente en equipos para cumplir con un objetivo común.
  3. Que el alumno sea capaz de diseño e implementación de servicios, aplicaciones web utilizando herramientas y lenguajes como HTML, CSS, JavaScript (incluyendo AJAX), back-end scripting y una base de datos, a un niverl intermedio.
  4. Desarrollar aplicaciones móviles, administrar servidores web en sistemas basados en UNIX y aplicar técnicas de seguridad en la web a un nivel intermedio.
